# 功能路线图

本文档记录 DoLike 项目的已完成功能与未来规划。

---

## v0.1（当前版本）

### 已完成

#### M0 — 文档与方向冻结
- [x] PRD 产品需求文档
- [x] 后端架构文档
- [x] 前端调整文档
- [x] 项目定位："本机使用的个人归档系统"

#### M1 — 本地账号与接入骨架
- [x] 本地账号注册、登录、退出登录
- [x] 修改密码
- [x] HttpOnly session cookie 会话保持
- [x] 三路抖音接入骨架：
  - 方案 A：CloakBrowser 无头浏览器扫码
  - 方案 B：手动 Cookie 粘贴导入
  - 方案 C：MV3 浏览器扩展桥接
- [x] Cookie 加密存储（scrypt + aes-256-gcm）

#### M2 — 归档主链路
- [x] 视频类内容基础抓取器：
  - 个人作品（user_post）
  - 喜欢（user_like）
  - 收藏视频（user_collect_video）
- [x] 归档调度服务（全量 / 增量自动判断）
- [x] 下载队列管理（worker 线程、并发控制、断点续传）
- [x] 下载队列的暂停、恢复、停止
- [x] 库查询（搜索 / 筛选 / 排序）
- [x] 媒体 Range 流式分发
- [x] WebSocket 进度推送

#### M3 — 离线浏览
- [x] "我的归档"页面：账号管理、视频网格展示
- [x] 视频子筛选：作品 / 喜欢 / 收藏 / 稍后再看
- [x] 长短视频筛选
- [x] 批量操作：全选、反选、下载、从视频中移除
- [x] 本地播放器（xgplayer）
- [x] 内容软删除（hidden 墓碑）

#### M4 — 音乐与合集
- [x] 音乐收藏后端归档链路
- [x] 合集后端归档链路
- [x] 音频提取功能（从视频中提取音频）

#### 扩展
- [x] MV3 浏览器扩展骨架
- [x] 四种 Cookie 采集方式（M1-M4）
- [x] 绑定账号 + 推送列表
- [x] 收藏夹、稍后再看的推送支持
- [x] Native Messaging 主机（Python）

#### 配套功能
- [x] Cookie 有效性测试
- [x] 意见反馈入口（GitHub Issues）
- [x] 前端账号面板状态轮询

## 优先级说明

| 优先级 | 标识 | 说明 |
| --- | --- | --- |
| P0 | 阻塞 | 核心功能不可用，必须立即修复 |
| P1 | 高 | 近期（v1.1）重点开发 |
| P2 | 中 | v1.2 范围 |
| P3 | 低 | v2.0 或更远 |

当前 v0.1 已知阻塞项（P0）：

- CloakBrowser 扫码路径 `secUid` 识别错误 → 导致归档内容归属到错误账号
- 扩展桥接链路待真实环境验证
